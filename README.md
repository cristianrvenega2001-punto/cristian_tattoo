## 📋 Descripción

Plataforma web para **Cristian Tattoo | Los Andes Ink**, servicio de tatuajes profesionales a domicilio operado por Cristian Venega con 28 años de experiencia. El sitio actúa como vitrina digital, canal de captación de clientes y herramienta de gestión de citas.

---

## ✨ Características

- 📱 **Diseño responsive** — optimizado para móviles con navegación fluida
- 🖼️ **Portafolio filtrable** — +40 trabajos reales organizados por estilo (Realismo, Blackwork, Fine Line, Full Color, Lettering, Watercolor)
- 📅 **Sistema de agenda online** — página dedicada `agenda.html` para reservas directas
- 🏆 **Promo FIFA World Cup 2026** — sistema de descuentos dinámico con modales de predicción de marcador integrados con Netlify Forms y WhatsApp
- 📝 **Guía de cuidados post-tatuaje** — contenido por temporada (verano/invierno/general) con tabs interactivos
- ⭐ **Sistema de reseñas** — formulario de testimonios integrado con Netlify Forms
- 🎨 **Portafolio con video** — sección de proceso en video con player HTML5
- 🔍 **SEO local optimizado** — Schema.org, Open Graph, sitemap.xml, robots.txt, geo meta tags orientados a búsquedas en Provincia de Aconcagua
- 📍 **Google Business verificado** — integración con Google Search Console
- 💬 **CTA WhatsApp directo** — botón flotante disponible en todas las secciones

```

├── index.html                   # Página principal

├── netlify.toml                 # Configuración de deploy
├── sitemap.xml                  # SEO sitemap

```
